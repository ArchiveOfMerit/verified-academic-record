# Search Discovery Suppression

This branch establishes a structured suppression framework for stale, disputed, broken, misattributed, and live third-party search and discovery artifacts.

It is designed to:
- retire or deindex controllable legacy assets,
- strengthen canonical public-reference signals,
- distinguish live source material from stale third-party artifacts,
- distinguish disputed, unverified, misattributed, and potentially hallucinated materials from authenticated source records,
- preserve technical findings about search-result persistence, dead routes, query distortion, degraded identity presentation, and third-party discoverability.

## Verification Boundary

This repository does not treat court-related, arrest-related, detention-related, or jail-related claims as verified merely because they appear in a search result, AI response, aggregator page, or third-party document. Where such material is disputed by the subject and has not been independently authenticated, it is classified as disputed, unverified, misattributed, or potentially hallucinated depending on source type.

This branch does not directly modify third-party search indexes or third-party hosted documents. It strengthens controllable signals, preserves technical records, and documents external artifacts for suppression tracking and classification.
