Local-only controls applied

  • Content-Security-Policy blocks all remote fetches, frames, workers, forms, fonts, media, and connections.
  • robots directives request no indexing, no archive, and no snippet generation.
  • referrer is set to no-referrer.
  • No external scripts, stylesheets, images, or embeds are loaded.
  • All runtime tokens and identifiers are replaced with placeholders.

Redaction summary

  • Personal name claims replaced with [REDACTED_NAME].
  • Email replaced with [REDACTED_EMAIL].
  • Phone/SMS fields replaced with [REDACTED_PHONE].
  • JWT tokens replaced with [REDACTED_TOKEN].
  • IP and request IDs replaced with local placeholders.
  • Viewer source, Chrome extension paths, and embedded PDF source references blocked.
