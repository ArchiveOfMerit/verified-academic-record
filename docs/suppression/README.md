# Search Discovery Suppression

This branch maintains a structured suppression, classification, and documentation framework for stale, disputed, broken, misattributed, and live third-party search and discovery artifacts.

It is designed to:

- document live third-party article, archive, listing, and index surfaces that continue to create non-canonical discovery pathways;
- retire, suppress, or deindex controllable legacy assets where action is available;
- strengthen canonical public-reference and academic-identity signals;
- distinguish live source material from stale, dead, broken, or residual third-party artifacts;
- distinguish disputed, unverified, misattributed, and potentially hallucinated materials from authenticated source records;
- preserve technical findings concerning search-result persistence, article-surface exposure, listing-surface exposure, dead routes, query distortion, degraded identity presentation, and third-party discoverability; and
- track incident-level evidence in both human-readable and machine-readable form.

## Verification Boundary

This branch does not treat court-related, arrest-related, detention-related, jail-related, or similar third-party claims as verified merely because they appear in a search result, AI response, aggregator page, archive article, listing surface, or third-party document.

Where such material is disputed by the subject and has not been independently authenticated, it is classified according to source condition and evidentiary status, including as disputed, unverified, misattributed, stale, non-canonical, or potentially hallucinated, as appropriate.

This branch does not directly modify third-party search indexes or third-party hosted documents. Instead, it strengthens controllable signals, preserves technical records, documents external artifacts for suppression tracking and classification, and distinguishes among:

- live article surfaces;
- live listing or index surfaces;
- stale residual fragments;
- dead or degraded routes; and
- non-authoritative third-party discovery patterns.
