# Search Discovery Suppression

This branch establishes a structured suppression framework for stale search and discovery artifacts.

It is designed to:
- retire or deindex controllable legacy assets,
- strengthen canonical public-reference signals,
- distinguish live source material from stale third-party artifacts,
- preserve technical findings about search-result persistence, dead routes, and query distortion.

This branch does not directly modify third-party search indexes or third-party hosted documents. It strengthens controllable signals and documents external artifacts for suppression tracking.
